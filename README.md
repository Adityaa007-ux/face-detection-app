# AI Face Detection System

## Project Overview
This project is an AI-based Face Detection Web Application developed using Python, OpenCV, and Streamlit.

The system detects human faces from uploaded images and highlights them using green rectangles.

---

## Features
- Upload image from computer
- Automatic face detection
- AI-based computer vision
- User-friendly web interface
- Real-time processing

---

## Technologies Used
- Python
- OpenCV
- Streamlit
- NumPy

---

## How to Run the Project

1. Install required libraries:
pip install -r requirements.txt

2. Run the application:
streamlit run app.py
